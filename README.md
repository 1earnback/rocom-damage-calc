# 洛克王国世界 对战伤害计算器

基于 Next.js 开发的洛克王国世界 对战伤害计算器，纯前端实现，支持实时计算精灵伤害。
灵感来[pokemon showdown 伤害计算器](https://calc.pokemonshowdown.com/)。
目前为初始测试版本，可能含有大量错误，欢迎在issue中反馈 bug。

## 功能特性

✅ **精灵图鉴系统**
- 支持 461+ 只精灵数据
- 支持精灵名称搜索
- 显示精灵属性、种族值、特性等信息

✅ **精灵定制系统**
- 25 种性格选择
- 全部六维个体值配置（支持勾选启用，默认值+10）
- 全部六维能力等级修正（支持+10%/-10%输入，自动合并 buff/debuff）
- 实时计算战斗属性

✅ **伤害计算系统**
- 完整的伤害计算公式
- 支持属性克制关系
- 支持本系加成
- 支持能力等级修正
- 支持战斗特性效果
- 显示详细伤害结果

✅ **技能系统**
- 489 个技能数据
- 支持技能搜索
- 自动显示攻击方能学会的技能
- 技能效果解析

✅ **特性系统**
- 支持战斗特性计算
- 向心力：前两个有伤技能威力 +30
- 凶煞：队伍存在恶系时，双攻 +50%
- 目空：非光系技能威力 +25%
- 专注力：入场首回合，物攻技能威力 +100%
- 魔法增效：魔攻技能威力 +70%
- 绝对秩序：受到非自身属性攻击时伤害 -50%

✅ **性能优化**
- Web Worker 支持大数据处理
- 数据缓存机制
- 工具函数支持（防抖、节流、懒加载）

✅ **移动端适配**
- 响应式设计
- 玻璃拟态 UI
- 触摸友好界面

## 技术栈

- **框架**: Next.js 14.2.5 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS + 玻璃拟态设计
- **数据源**:
  - 精灵数据: /data/sprites.json (~10MB)
  - 技能数据: /data/skills_all.csv (~47KB)
- **包管理**: pnpm

## 开始使用

### 安装依赖

```bash
cd website
pnpm install
```

### 创建数据文件链接

数据文件通过软链接到 public 目录：

```bash
ln -s /home/user/lkwg_tools/data/sprites.json /home/user/lkwg_tools/website/public/data/sprites.json
ln -s /home/user/lkwg_tools/data/skills_all.csv /home/user/lkwg_tools/website/public/data/skills_all.csv
```

### 启动开发服务器

```bash
pnpm dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
pnpm build
pnpm start
```

## 伤害计算公式

```
单次伤害 = (攻击裸值 / 防御裸值) × 0.9 × 有效威力 × 能力等级 × 本系加成 × 克制关系 × 天气影响 × 减伤系数
最终伤害 = 单次伤害 × 连击次数
```

### 各项说明

- **攻击裸值**: 物攻或魔攻（根据技能类型）
- **防御裸值**: 物防或魔防（根据技能类型）
- **有效威力**: 技能威力 × 应对倍率 + 威力加成
- **能力等级**: (1 + 攻击提升 + 敌方防御降低) / (1 + 攻击降低 + 敌方防御提升)
- **本系加成**: 1.5（技能属性与精灵属性匹配）或 1.0
- **克制关系**: 单属性 2x/0.5x，双弱点 3x，双抵抗 0.25x
- **天气影响**: 雨天水系技能 1.5x

## 精灵属性计算

```
HP = [1.7 × 种族值 + 个体值 × 6 × 0.85 + 70] × (1 + 性格修正) + 50
其他属性 = [1.1 × 种族值 + 个体值 × 6 × 0.55 + 10] × (1 + 性格修正) + 50
```

## 性格系统

25 种性格，每种影响两个属性：
- 最高种族值属性: +20%
- 最低种族值属性: -10%
- 其余属性: 不变化

## 开发说明

### 数据加载

- sprites.json 较大（~10MB），首次加载可能需要几秒钟
- 数据在客户端加载，无需服务器支持

### 性能考虑

- 使用了懒加载和缓存机制
- 精灵和技能数据只在页面加载时获取一次
- 计算结果实时更新，无需刷新页面

### 移动端适配

- 响应式设计，支持移动设备
- 使用 Tailwind CSS 的响应式类
- 玻璃拟态设计提供良好的视觉效果
