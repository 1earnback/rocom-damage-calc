# 洛克王国世界 对战伤害计算器
一个基于 Next.js 开发的纯前端《洛克王国:世界》对战伤害计算器。思路源于[pokemonshowdown伤害计算器](https://github.com/showdownjs/showdown)。

## 🛠 技术栈

- **框架**：[Next.js](https://nextjs.org/) (App Router)
- **样式**：原生 CSS (Global CSS) + 玻璃拟态设计
- **数据源**：
  - 精灵图鉴数据：../data/sprites.json (约 10MB)
  - 技能数据库：../data/skills_all.csv (约 47KB, 489 个技能)

- **部署**：GitHub Actions + GitHub Pages

## 开发任务

### 阶段一：基础功能 (MVP)

#### 1. 数据加载与图鉴系统
- 加载 `sprites.json` 精灵数据
- 加载 `skills_all.csv` 技能数据
- 实现精灵列表展示与联想搜索（支持中文名搜索）
- 实现技能列表与搜索功能
- 支持精灵详情展示（种族值、属性、特性、可学技能等）

#### 2. 精灵定制系统
- **性格选择**：25 种性格选择器，显示性格对属性的影响（+20% / -10%）
- **个体值配置**：支持勾选启用六维个体值（默认值+10，范围0-10），只显示勾选的属性输入框
- **Buff/Debuff 配置**：合并为单一能力等级修正输入框，支持正负值（+10%提升/-10%降低）。
- **实时属性计算**：根据配置实时更新精灵的战斗属性

#### 3. 基础伤害计算器
- **攻击方配置**：选择精灵、配置性格/个体值/buff
- **防守方配置**：选择精灵、配置性格/个体值/buff
- **技能选择**：选择攻击方使用的技能（从技能库中选择或手动输入）
- **伤害计算**：实现基础伤害公式
  - 攻防裸值计算
  - 有效威力计算（技能威力 + 威力加成）
  - 能力等级计算（boost/reduce 修正）
  - 本系加成（STAB）
  - 属性克制（单属性 2x/0.5x，双属性特殊规则）
- **结果展示**：显示最终伤害值、伤害百分比

## 核心计算公式

### 精灵属性计算
```
HP = [1.7 × 种族值 + 个体值 × 6 × 0.85 + 70] × (1 + 性格修正) + 50
其他属性 = [1.1 × 种族值 + 个体值 × 6 × 0.55 + 10] × (1 + 性格修正) + 50
```

### 伤害计算公式
```
单次伤害 = (攻击裸值 / 防御裸值) × 0.9 × 有效威力 × 能力等级 × 本系加成 × 克制关系 × 天气影响 × 减伤系数
最终伤害 =单次伤害 × 连击次数

其中：
- 有效威力 = 技能威力 × 应对倍率 + 威力加成
- 能力等级 = (1 + 我方攻击提升 + 敌方防御降低) / (1 + 我方攻击降低 + 敌方防御提升)
- 本系加成 = 1.25（技能属性与精灵属性匹配）或 1.0
- 克制关系 = 单属性 2x/0.5x 或双属性特殊规则（双弱点 3x，双抵抗 0.33x）
```

## 开发注意点

1. **代码转换**：将 Python 的 `sim/damage_calc.py`、`sim/pokemon_db.py` 等模块的纯计算逻辑转换为 TypeScript/JavaScript
2. **数据格式**：
   - sprites.json 为 JSON 格式，可以直接在前端加裁
   - skills_all.csv 为 CSV 格式，需要在前端解析
3. **性能优化**：
   - sprites.json 约 10MB，考虑使用 Web Worker 或分块加载
   - 技能数据库较小，可直接加载到内存
4. **类型安全**：使用 TypeScript 定义精灵、技能、属性等类型
5. **包管理**：使用 pnpm 作为包管理器
6. **Python 环境**：如需运行 Python 代码验证，使用 uv 和 venv 虚拟环境
